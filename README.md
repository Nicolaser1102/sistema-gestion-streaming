# Sistema de Gestión de Streaming

Proyecto académico desarrollado para la asignatura **Programación Orientada a Objetos**.

## Descripción
El sistema simula el funcionamiento básico de una plataforma de streaming, permitiendo la gestión de usuarios, catálogo de contenidos, reproducción multimedia y control de acceso mediante suscripciones.

## Módulos del sistema
- Autenticación y gestión de usuarios
- Catálogo de contenidos
- Reproducción de contenidos
- Gestión de suscripciones
- Administración del catálogo

## Tecnologías
- Lenguaje: Go
